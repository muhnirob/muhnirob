## 📖 What is Nirob Digital?

**Nirob Digital** is a personal documentation project and a real-world journey.  
It tracks the step-by-step process of building multiple online income streams using:

- 🌐 **Web3 & Crypto** (cWallet, Coinbase, Binance, DeFi staking)
- 🤖 **AI Content Creation** (HeyGen, PixVerse, Seedance, Runway ML)
- ⚙️ **Automation** (n8n, GitHub Actions, Zapier)
- 🛒 **E-commerce & Digital Products** (Print on Demand, Gumroad, Shopify)
- 🔐 **Networking & Security** (Cloudflare, Tailscale, penetration testing tools)

This repository is **my public roadmap, toolkit, and accountability partner**.
